## Command Line Interface(CLI)
CLI get command from a user. The user can enter '.md' filename, '.html' filename and 'option'.
By the user input, 

###Style(option)
1. plain: plain style(default)
> 

2. fancy: fancy style
> --fancy

3. slide: slide style
> --slide
